// Introduction

Présentation du projet

Le projet Cinéma est une application web développée en PHP et MySQL dans le cadre du BTS SIO, réalisée chez AITech. L'objectif est de permettre à tout internaute de consulter une liste de films avec leurs informations associées : titre, description, date et producteur.

L'application repose sur une base de données MySQL fournie initialement, hébergée sur une machine distante. Le projet suit une architecture MVC, séparant la logique métier (modèle), la présentation (vue HTML/CSS) et les interactions (contrôleur PHP).

Il couvre les compétences SLAM : conception et développement d'une solution applicative, gestion des données et connexion à une base distante.

// Déroulement

Étapes du développement

Phase 1
Analyse & prise en main
Lecture du cahier des charges, découverte de la base de données fournie (7 tables), identification des besoins fonctionnels.
Phase 2
Mise en place de l'environnement
Configuration des deux machines virtuelles (serveur web + serveur BDD), paramétrage de la connexion réseau distante.
Phase 3
Architecture MVC & modèles
Structuration du projet en MVC, développement des modèles PHP (requêtes MySQL via PDO) pour Film, Artiste, Genre, Pays, Notation.
Phase 4
Développement des vues
Création des pages HTML/CSS : liste des films, page de détail, affichage dynamique des données récupérées depuis la BDD.
Phase 5
Tests & résolution des difficultés
Résolution de la connexion à la BDD distante, tests fonctionnels, vérification de l'affichage correct de toutes les informations.
